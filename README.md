# Healthcare App

This repository contains a cross-platform healthcare application built with React, Vite, and Capacitor. The project provides features for users to manage health-related data such as reports, BMI calculations, blood donors, and more. It integrates with several backend services for storage and authentication.

## 🚀 Tech Stack

- **Frontend:** React 19, Vite
- **Mobile & Desktop:** Capacitor (Android & iOS)
- **Storage & Authentication:**
  - AWS S3
  - Firebase
  - Supabase
  - Cloudinary
- **Routing:** React Router DOM
- **Linting:** ESLint

## 🔧 Features

- User signup/login with Firebase Authentication
- Upload and view medical reports
- BMI calculator
- Blood donors directory
- Dashboard with secure routes
- Integration with AWS S3, Supabase and Cloudinary for file storage

## 🛠️ Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```

4. **Working with Capacitor**
   - Add a platform: `npm run cap:add ios` or `npm run cap:add android`
   - Sync after changes: `npm run cap:sync`
   - Open platform project:
     - `npm run cap:open:ios`
     - `npm run cap:open:android`
   - Build for devices:
     - `npm run cap:build:ios`
     - `npm run cap:build:android`

## 📘 Configuration

Edit the files inside `src/config/` to provide your own credentials for Firebase, AWS, Supabase, and Cloudinary.
